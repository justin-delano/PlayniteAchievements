Drop bundled notification sounds here as .wav files.

These files are packaged into the built extension automatically via PlayniteAchievements.csproj.

Recommended usage:
- Keep file names short and stable.
- Use PCM .wav files for best compatibility.
- After adding files here, rebuild the extension so they are copied into the packaged output.

At runtime, Local settings will list any .wav files found under Resources\Sounds.
